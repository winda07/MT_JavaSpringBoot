SECURITY

fitur security dalam aplikasi menggunakan mekanisme Authentication dan Authorization.
Authentication adalah fitur security yang menangani siapa user yang masuk kedalam sistem,
Authorization adalah setelah user tersebut berhasil masuk kedalam system lalu user tsb
bisa mengakses kemana saja.
